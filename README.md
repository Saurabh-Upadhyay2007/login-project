📌 Project Description

This is a web-based login system developed using HTML, CSS, PHP, and MySQL.
It allows users to register, log in securely, access a dashboard, and log out.

---

🚀 Features

- ✅ User Registration
- ✅ User Login Authentication
- ✅ Password Storage in Database
- ✅ User Dashboard
- ✅ Logout System
- ✅ Clean UI using CSS / Bootstrap

---

🛠️ Technologies Used

- HTML
- CSS
- PHP
- MySQL

---

📂 Project Structure

login_project/
│── index.php        (Login Page)
│── register.php     (Register Page)
│── dashboard.php    (User Dashboard)
│── logout.php       (Logout System)
│── db.php           (Database Connection)
│── style.css        (Styling)

---

⚙️ How to Run (Local Setup)

1. Install XAMPP
2. Move project folder to:
   C:\xampp\htdocs\
3. Start Apache and MySQL
4. Open browser and go to:
   http://localhost/login_project
